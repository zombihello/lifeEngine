## Description
LifeEngine is a 3D game engine inspired by Source Engine and Quake Engine 

## Screenshots
This brunch is fully rework the engine and currently not have any screenshots

## Building

### Windows
1. Install [Visual Studio 2022 Community](https://visualstudio.microsoft.com/downloads/).
2. Run `src/generate_solution.bat` with parameter ```-game <gameName>```. By default this script generate project for demo game <b>'eleot'</b>.
3. Open the solution created in `src/<gameName>.sln`.
4. Build project `launcher`.
